# 1. Introduction to Artificial Intelligence

Artificial Intelligence (AI) is a branch of computer science that enables machines to perform tasks that normally require human intelligence.

Examples:

- Speech Recognition
- Face Recognition
- Language Translation
- Recommendation Systems
- Chatbots

---

# 2. What is Generative AI?

Generative AI (GenAI) is a type of Artificial Intelligence that can create new content such as:

- Text
- Images
- Videos
- Audio
- Code
- Music

Instead of only analyzing existing data, Generative AI learns patterns from large datasets and generates new content.

---

# 3. History of Generative AI

- **1950** – Alan Turing introduced the Turing Test.
- **1956** – Artificial Intelligence became a research field.
- **1997** – IBM Deep Blue defeated world chess champion Garry Kasparov.
- **2014** – Generative Adversarial Networks (GANs) were introduced.
- **2017** – Transformer architecture was introduced.
- **2020 onwards** – Large Language Models such as GPT became widely used.

# 7. Machine Learning and Deep Learning

## Machine Learning (ML)

Machine Learning enables computers to learn from data without being explicitly programmed.

Examples:

- Email Spam Detection
- Product Recommendation
- Fraud Detection

---

## Deep Learning (DL)

Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with many layers.

Applications:

- Image Recognition
- Speech Recognition
- Language Translation
- Chatbots

---

# 8. Neural Networks

Components:

- Input Layer
- Hidden Layer(s)
- Output Layer

# 9. Large Language Models (LLMs)

Large Language Models are AI models trained on massive amounts of text data.

Examples:

- GPT
- Gemini
- Claude
- Llama

Capabilities:

- Answer questions
- Summarize text
- Translate languages
- Generate code
- Write emails
- Create stories

# 10. Foundation Models

Foundation Models are large AI models trained on diverse datasets that can be adapted to many different tasks.

Examples:

- GPT
- Gemini
- Llama

Benefits:

- Reusable
- General-purpose
- Fine-tunable

# 11. Transformer Architecture

Transformers are deep learning models introduced in 2017.

Features:

- Attention Mechanism
- Parallel Processing
- Context Understanding
- Better Language Generation

Benefits:

- Faster Training
- Higher Accuracy
- Better Context Handling

# 12. Prompt Engineering

Prompt Engineering is the process of writing effective instructions to obtain better AI-generated responses.

Example Prompt:

```text
Explain Docker in simple words with an example.
```

Good Prompt Characteristics:

- Clear
- Specific
- Detailed
- Context-aware

---

# Types of Prompts

### Zero-shot Prompt

No examples are provided.

Example:

```text
Translate "Hello" into French.
```

---

### One-shot Prompt

One example is provided.

---

### Few-shot Prompt

Multiple examples are provided before asking the model to perform a task.
